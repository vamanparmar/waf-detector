# 🛡️ WAF / IDS / IPS Detector

Detect Web Application Firewalls via multi-probe HTTP fingerprinting.

## 🚀 How to Run

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/waf-detector.git
cd waf-detector/home/claude/waf-v2/backend
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Start backend
```bash
python app.py
```

### 4. Open frontend
Open `home/claude/waf-v2/frontend/index.html` in your browser.
